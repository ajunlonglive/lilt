**Required PHP Version**

- PHP 7.1+

**Installation**

```bash
$ composer require lilt/lilt
$ echo "extension=lilt.so" >> /path/to/php.ini

or clone then

$ composer install
$ echo "extension=lilt.so" >> /path/to/php.ini
```

**Benchmark**

```bash
$ composer bench
```

**Tests**

```bash
$ composer test
$ composer memtest
```
